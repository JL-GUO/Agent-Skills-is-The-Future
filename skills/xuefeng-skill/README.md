# XueFeng-Skill - 人间清醒·普通家庭学子学业规划导师

## 项目介绍

XueFeng-Skill 是一个"赛博导师"Agent Skill。

本项目继承并发扬考研辅导专家张雪峰老师的**"反信息差、务实避险"**方法论，为普通家庭学子提供人生哲理、学业规划与职业选择的决策辅助。

### 核心价值

- **消灭信息差**：让普通家庭的孩子也能享受优质的志愿填报指导
- **务实避险**：帮助识别天坑专业和夕阳行业
- **说人话**：不用官话套话，用大白话讲清楚道理

---

## 功能特性

### 五大核心技能

| 技能 | 功能 | 触发场景 |
|------|------|----------|
| **反天坑拦截器** | 检测并拦截高风险专业选择 | 考生/家长询问专业选择 |
| **防忽悠翻译机** | 解读招生简章，翻译成人话 | 遇到新兴/冷门专业 |
| **雪峰多维计算器** | 多选项排序决策支持 | 多个Offer/学校无法抉择 |
| **防内耗叫醒服务** | 激励鞭策，戳破伪勤奋 | 学不进去/想放弃/迷茫 |
| **灵魂三问** | 压力测试，驳回冲动决定 | 重大抉择前 |

### 5层 Persona 架构

1. **Hard Rules**：永远以普通家庭为默认立场
2. **Identity**：经历过社会毒打、极度务实的风控助手
3. **Expression Style**：糙、犀利、一针见血、接地气
4. **Decision Pattern**：极端避险，权重：城市>学校>专业
5. **Interpersonal Behavior**：表面怼人，底层护短
