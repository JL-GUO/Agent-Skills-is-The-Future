# 微信个性化祝福生成示例

## 示例1：工作伙伴（产品经理）

**用户输入**：
> 帮我给产品经理老张发个新年祝福，幽默一点的

**聊天记录分析**：
- 总消息数：234条
- 高频话题：需求变更、项目排期、上线延期、Bug修复
- 情感关键词：辛苦、加油、顶住、理解
- 关系类型：工作伙伴

**生成祝福**：

```
【联系人】产品经理老张
【关系分析】工作伙伴，近期频繁讨论项目上线和需求变更
【个性化祝福】
张哥，蛇年大吉！过去一年咱们一起扛过了无数次"需求小改"和"明天上线"，
革命友谊比代码还坚固。新的一年，祝你负责的项目全绿灯、上线零Bug，
产品经理不拍脑袋，程序员不甩锅，咱们继续愉快合作！🐍🧧
```

## 生成原则总结

### 1. 关系特征体现

| 关系类型 | 祝福侧重点 |
|---------|-----------|
| 工作伙伴 | 事业顺利、项目成功、合作愉快 |
| 家人 | 健康平安、少操心、多陪伴 |
| 好友 | 开心快乐、共同爱好、兄弟情谊 |
| 客户 | 感谢支持、期待合作、正式得体 |

### 2. 话题融入技巧

- **高频话题**：在祝福中自然提及（如"需求变更"、"开黑"）
- **情感关键词**：呼应聊天氛围（如"辛苦"→"注意休息"）
- **具体细节**：提及具体事件增加真实感

### 3. 语气调整

- **幽默**：用梗、自嘲、轻松的调侃
- **温馨**：关心、感恩、表达思念
- **正式**：礼貌、专业、简洁
- **简短**：精炼、有力、适合快速发送

### 4. 避免的雷区

- ❌ 模板化："祝你新年快乐，万事如意"
- ❌ 过于笼统：没有体现任何关系特征
- ❌ 过度亲密：对客户或不太熟的人用太随意的语气
- ❌ 负面暗示：避免提及不愉快的事情
